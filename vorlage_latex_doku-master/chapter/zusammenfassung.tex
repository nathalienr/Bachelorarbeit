\chapter{Summary}
\label{cha:summary}

The following points should be addressed on two to three pages:

\begin{itemize}
	\item Which objective was to be achieved
	\item Which approach was chosen
	\item What was achieved; state key results, preferably using quantitative metrics
	\item Could the results, after critical evaluation, contribute to achieving the goal or solving the problem
	\item Outlook
\end{itemize}

The summary must include clear statements regarding the outcome of the work. Usually, results can be described not only qualitatively but also quantitatively, e.g., \glqq \ldots an efficiency increase of \SI{12}{\percent} was achieved.\grqq~or \glqq \ldots the testing duration was reduced by \SI{2}{\hour}.\grqq

The results presented in the summary should naturally relate to the questions and objectives defined in the introduction.
